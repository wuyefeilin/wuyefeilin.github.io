# anaconda

# 安装

下载安装包（https://www.anaconda.com/distribution/） 安装

# 命令

conda create -n 名称 conda python=3.7,创建python3.7名为“名称”的虚拟环境 conda remove -n 名称 --all， 删除虚拟环境 conda activate 名称， 激活环境 conda deactivate, 退出环境

conda info --env: 查看安装的虚拟环境变量

完
